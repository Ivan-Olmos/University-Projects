## 📊 Projects Overview

This subject focused on data analysis, statistics, and business insights using real-world datasets.

## 📈 Project 1: Property Data Analysis (Orange, NSW)

### 🧩 Objective
Analyse real estate data to extract insights that could support decision-making for a real estate agent.

---

### 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Charts, Analysis ToolPak)

---

### ⚙️ What I did
- Cleaned and structured raw data (108 records, 11 variables) :contentReference[oaicite:0]{index=0}  
- Converted units (hectares to square meters)  
- Handled missing data (e.g. parking values)  
- Created pivot tables and charts (bar chart, pie chart)  
- Generated descriptive statistics using Excel  
- Built histogram to analyse price distribution  
- Analysed relationships between variables (bedrooms vs price)  

---

### 📊 Key Findings
- ~93.5% of properties were classified as houses :contentReference[oaicite:1]{index=1}  
- Property prices showed **right skewness** (few high-value properties)  
- Median is a better indicator than mean due to outliers  
- Moderate positive correlation between bedrooms and price (~0.56) :contentReference[oaicite:2]{index=2}  

---

### 🧠 What I learned
- How to work with real-world messy data  
- Data cleaning and preprocessing techniques

- Turning data into business insights

## 📊 Project 2: Probability and Statistical Inference

### 🧩 Objective
Apply probability models and statistical inference to real-world business scenarios.

---

### 🛠️ Tools Used
- Excel (statistical functions, NORM.DIST, NORM.INV)

---

### ⚙️ What I did
- Built probability models (tree diagrams)  
- Calculated conditional probabilities  
- Analysed independence of variables  
- Worked with normal distributions (mean = 75g, SD = 4g) :contentReference[oaicite:3]{index=3}  
- Calculated probabilities of defects and compliance  
- Evaluated manufacturing risks using statistics  
- Built confidence intervals for defect rates  

---

### 📊 Key Findings
- ~1.7% probability of defective product below tolerance :contentReference[oaicite:4]{index=4}  
- Bulk packs had extremely low failure probability (~0.01%) :contentReference[oaicite:5]{index=5}

---

### 🧠 What I learned

- Understanding distributions and variability  
- Risk analysis using statistics

- How data supports decision-making
